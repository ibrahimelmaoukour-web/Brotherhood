# 🏛️ Brotherhood — Primitive Blueprint Guide

> **A collaborative, open-source reference for primitive crafting techniques. Anyone can join, learn, and contribute.**

## 🎯 Mission

Brotherhood is a community-driven resource documenting the progression of human civilization through primitive crafts — from discovering water sources to building mechanical and electrical systems. It's built **by the community, for the community**.

## 🚀 Quick Start

**Want to view the guide?** Open `bortherhood/index.html` in your browser.

**Want to contribute?** See [CONTRIBUTING.md](CONTRIBUTING.md) — we welcome all skill levels!

## 💡 What's Inside

Brotherhood covers 5 major civilization stages:

1. **🚰 Sacred Water & Soil** — Finding water and basic agriculture
2. **🔥 The Foundation of Fire** — Mastering fire and ceramics
3. **⚒️ The Pyrotechnic Age** — Smelting metals and advanced crafting
4. **⚙️ The Mechanical Bridge** — Converting natural forces to machines
5. **⚡ The Electromagnetic Leap** — Harnessing electricity

Each stage includes detailed craft roadmaps with techniques, resources, outputs, and dependencies.

## 🤝 How to Contribute

This project is **100% open** to contributions:

- **Add new crafts** following the existing format
- **Expand existing stages** with more details
- **Fix errors or improve clarity** in existing entries
- **Add resources, videos, or diagrams**
- **Suggest new civilization stages**

See [CONTRIBUTING.md](CONTRIBUTING.md) for detailed instructions.
